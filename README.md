# Acaply

A social media website for high schoolers.

![Logo](/public/images/logo.png)

### Technologies:

- Database: MongoDB, Mongoose.
- Website: HTML, CSS, Bootstrap, Embedded JavaScript Templates.
- Language: JavaScript.
- Email: SendGrid
- Other: Axios, Express, Node.js.
