# Acaply

A social media website for high schoolers.

### Technologies:
- Database: MongoDB, Mongoose.
- Website: HTML, CSS, Bootstrap, Embedded JavaScript Templates.
- Language: JavaScript.
- Other: Axios, Express, Node.js.

### Usage:
- `git clone https://github.com/pillious/acaply.com.git`.
- `cd acaply.com`.
- `npm install`.
- `npm start`.
